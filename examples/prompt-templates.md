# Prompt 模板参考

## 角色锚点 Prompt 模板（Z-image Turbo）

```
Character reference board, photorealistic, one single Chinese [性别], [年龄] years old, [身份/职业]. Four panel layout on clean light gray background: Top-left panel shows full body standing front view, [服装描述]. Top-right panel shows close-up headshot, [面部特征]. Bottom-left panel shows three-quarter side profile, same clothing, same man. Bottom-right panel shows pure side profile. All panels show the exact same consistent person, identical face and clothing. Professional studio lighting, 8k hyperrealistic DSLR photography, no text no labels, pure visual reference.
```
